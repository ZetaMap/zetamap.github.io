def process_died(self, process_name, exit_code): ¶

Notifies listener that process has died. This callback only occurs for processes that die during normal process monitor execution -- processes that are forcibly killed during ProcessMonitor shutdown are not reported. @param process_name: name of process @type process_name: str @param exit_code: exit code of process. If None, it means that ProcessMonitor was unable to determine an exit code. @type exit_code: int
